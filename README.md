# Kurek Ashley — Canonical Repository

This repository contains the canonical, source-faithful extraction of teachings derived from coaching sessions and transcript-based material.

## Purpose

To establish a structured, machine-readable, and human-legible body of work that preserves:

- Core language
- Repeated principles
- Observed patterns
- Emerging frameworks (unverified)

This repository is not a book, program, or interpretation layer.

It is the **source-grounded authority layer**.

## Scope

Included:
- Canonical phrases (verbatim)
- Design principles (cause–effect relationships)
- Concept index
- Framework candidates (unverified)

Excluded:
- Imposed systems or doctrines
- Marketing or narrative structures
- Certification or curriculum design
